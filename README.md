# Agency-in-a-Box

Fully autonomous Dutch e-commerce lead-to-revenue system. An AI-powered agency that operates autonomously, handling everything from lead research and code generation to PR creation and deployment testing.

## Project Structure

### Core Worker (`/src`)
- `config.py` — Centralised, secure configuration and Doppler secrets loader.
- `schemas.py` — Shared Pydantic models acting as the single source of truth across agents.
- `worker_core.py` — Main job queue consumer, heartbeat thread, budget reconciliation, and audit logging.
- `gsd_executor.py` — The "Get Shit Done" engine powered by Claude 3.5 Sonnet (Research & Implementation phases).
- `critic_agent.py` — Cross-model QA reviewer powered by Gemini 1.5 Pro. Evaluates the code generated by Claude.
- `telegram_bot.py` — Interfaces with the Telegram API for sending alerts and handling manual approval gates.
- `sandbox.py` — Orchestrates Fly.io ephemeral microVMs for secured code execution and test validation.
- `pr_creator.py` — GitHub integration layer that creates branches, commits code, and opens Pull Requests.
- `lead_poller.py` — Background daemon tracking unread database alerts and forwarding to Telegram.

### DMZ (Demilitarized Zone) (`/dmz`)
- `main.py` — A FastAPI web server handling external webhooks (Telegram callbacks) and forwarding actions securely to the core via database changes without direct core exposure.

### Trusted Execution Sandbox (`/fly-sandbox`)
- `Dockerfile` & `entrypoint.sh` — The environment and execution script for running untrusted, AI-generated Python code safely.

### Database Operations (`/sql`)
- `001_schema.sql` — Core tables definitions (projects, jobs, budget, audit).
- `002_watchdog.sql` — Queries to reclaim stalled jobs.
- `003_gdpr.sql` — Scheduled deletion logic for GDPR compliance.
- `004_pg_cron.sql` — Setup for PgCron scheduled routines.

## Quick Start
To get the system running locally or securely deployed to production on Fly.io, refer to the [Deployment Guide](DEPLOYMENT.md). The process takes roughly 45–90 minutes depending on API key acquisition.
